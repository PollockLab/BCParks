# BC Parks: Sampling priority index

This repository contains code to calculate spatial, taxonomic, and purpose-oriented scores to generate sampling priority index maps for BC Parks. 

[![badge](https://img.shields.io/static/v1?style=for-the-badge&label=VIEW&message=DEMOMAP&color=61bf75)](https://pollocklab.github.io/BCParks/scoremaps_inatamphibians.html) [![badge](https://img.shields.io/static/v1?style=for-the-badge&label=VIEW&message=NOTES&color=619dbf)](https://docs.google.com/presentation/d/1g8PiHl67avwK4GH8wkR0pne9CsFpAa7zvCzfNpNzPBU/edit?usp=sharing)

## Objective & approach

- Predict hotspots for next year's sampling
